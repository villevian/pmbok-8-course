Що таке
проєкт?

Визначення · Definition
Проєкт (Project) — це тимчасова ініціатива в унікальному контексті, що здійснюється для створення цінності.
A project is a temporary initiative in a unique context undertaken to create value. — PMBOK® 8

01
⏱️
Тимчасовий характер
Temporary Nature

Проєкт має чіткий початок і кінець. Він завершується коли досягнуто цілей, вичерпано ресурси або змінились пріоритети.

02
🌐
Унікальний контекст
Unique Context

Кожен проєкт відрізняється від інших — навіть якщо цілі схожі. Цілі, ризики, команда, середовище — все унікальне.

Унікальні цілі Унікальна команда Унікальне середовище
03
💎
Створення цінності
Value Creation

Проєкт переводить організацію з поточного стану в майбутній стан, створюючи відчутну або невідчутну цінність.
